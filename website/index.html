Welcome to Annotald

Annotald is a free and open-source program for annotating parsed corpora in the spirit of the Penn Historical Corpora of English. Annotald was originally written by Anton Ingason as part of the Icelandic Parsed Historical Corpus project. It is currently being maintained by Aaron Ecay at the University of Pennsylvania Linguistics department.

You may wish to:

  • Use Annotald
  • Consult the documentation
  • Visit Annotald on Github (including to file a bug report)

Annotald development has been funded by the following funding sources, which the developers gratefully acknowledge:
